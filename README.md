# Finance Visualizer

## Features
- Stage 1: Transaction CRUD, Monthly Chart
- Stage 2: Categories, Pie Chart, Dashboard
- Stage 3: Budget Tracking, Comparison Chartst

## Setup
1. Clone repo
2. `npm install`
3. Create `.env` with MongoDB URI
4. `npm run dev`
